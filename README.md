# Mobile Sales Dashboard

## Project Overview
This Power BI dashboard analyzes mobile sales performance across different cities, brands, payment methods, and time periods.

## Tools Used
- Power BI
- Excel

## Features
- Total Sales
- Total Quantity
- Transactions Analysis
- City-wise Sales Analysis
- Brand-wise Sales Analysis
- Customer Ratings Analysis
- Payment Method Analysis
